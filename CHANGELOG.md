# Changelog

All notable changes to this project will be documented in this file.

This project follows a lightweight semantic versioning convention while it is in early development. The Skill version is stored in `SKILL.md` as `metadata.version`.

## 0.1.1 - 2026-05-31

### Added

- Add `version` command to show the local Skill version.
- Add `check-update` command to compare the local version with the GitHub main branch.
- Document upgrade workflow for Skill users and release workflow for maintainers.

### Changed

- Update README for open-source usage and version management.

## 0.1.0 - 2026-05-31

### Added

- Initial CowAgent Skill for convertible bond subscription and listing reminders.
- Support configurable calendar JSON data source.
- Support subscription queries, listing-date queries, listing tracking, scheduler task creation, reminder listing, and status inspection.
